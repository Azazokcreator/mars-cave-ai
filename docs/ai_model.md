\# Модель искусственного интеллекта



---



\## Используемый датасет



CaveSeg Dataset



Содержит:



* реальные изображения пещер
* пиксельную сегментацию на 14 классов



Преобразование классов



Для задачи навигации:



14 классов были объединены в 2 категории:



SAFE:

&nbsp;

* открытая поверхность
* пол
* доступные зоны



OBSTACLE:



* стены
* камни
* узкие проходы



---



\## Архитектура модели





Используется:



UNet + ResNet18 encoder



Причины выбора:



* высокая точность сегментации
* быстрая работа
* подходит для real-time

Dataset: CaveSeg
