## Веса модели (Model Weights)


### Как скачать и подключить модель

Файл весов модели **не хранится в репозитории** (чтобы репо было лёгким и удобным для скачивания).

1) Скачайте веса по ссылке:  
**unet_safe_obstacle1.pth** → *https://drive.google.com/drive/folders/1Kk8PIlYy8znVffMW8rUNMkm53LjZ2R-z?usp=sharing*

2) Создайте папку (если её нет):

mkdir models

Поместите файл в:

models/unet_safe_obstacle1.pth

Проверьте структуру проекта:

mars-cave-ai/
  models/
    unet_safe_obstacle1.pth

Если путь к модели задан в коде, он должен указывать на этот файл.

The model weights are not stored in this repository to keep it lightweight and easy to clone.
